# 欢迎来到我的博客

这是一篇测试文章，用于演示如何在 Hexo 中使用 Aurora 主题发布文章。

## 文章特性

这篇文章展示了以下功能：

- **Markdown 语法支持**：可以使用标准的 Markdown 语法编写文章
- **代码高亮**：支持代码块语法高亮
- **封面图片**：可以在文章顶部显示封面图
- **分类和标签**：可以给文章添加分类和标签

## 代码示例

下面是一个简单的代码示例：

  console.log('欢迎使用 Aurora 主题');

## 结语

现在你可以开始编写自己的文章了！记得：

- 使用 `hexo new "文章标题"` 创建新文章
- 编辑 Markdown 文件
- 运行 `hexo clean && hexo g` 重新生成
- 使用 `hexo server -p 4002` 预览效果

祝你写作愉快！🎉
